# AGENTS.md - Guidelines for AI Assistants

## Rules

1. **Do NOT commit Markdown files** (`.md` files)
   - Documentation files should not be committed to the repository
   - This includes `README.md`, `CHANGELOG.md`, `AGENTS.md`, etc.

2. **Use English for all code comments**
   - All inline comments, docstrings, and code documentation must be in English
   - Example: `// Initialize the server` ✅
   - Example: `// 初始化服务器` ❌

## Why?

- **No Markdown commits**: Keep the repo focused on code. Documentation is generated or maintained separately.
- **English comments**: Ensures the codebase is accessible to all developers and AI tools globally.

---

_Follow these rules in all commits and code changes._
